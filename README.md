Реализуйте компонент <Card> так, чтобы можно составлять такую структуру:

<pre><Card>
<Card.Body>
<Card.Title>CardTitle</Card.Title>
<Card.Text>CardText</Card.Text>
</Card.Body>
</Card></pre>

Получившийся HTML:

<pre><div class="card">
<div class="card-body">
<h4 class="card-title">CardTitle</h4>
<p class="card-text">CardText</p>
</div>
</div></pre>
